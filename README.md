# Piano note player

## Description
Primitive piano that detects inputed keys and plays the corresponding notes.
## How to use
The program is set to detect certain keyboard letters(C, D, E, F, G, A, B) as their respective piano notes.
Input one of these letters and the program will play it's associated sound, or 'Q'/'q' to exit.
